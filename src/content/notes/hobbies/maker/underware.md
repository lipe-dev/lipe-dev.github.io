# Underware

Under-desk cable and connectivity management system. Part of the [[Gaming PC]] / office desk organization project.

## The Problem

Cable hell under the desk:
- Power cables
- USB connections
- Audio cables
- Display cables
- Network cables
- Charging cables

All tangled, all ugly, all in the way.

## The Solution

Underware is a 3D printed under-desk mounting system for:
- Cable routing and management
- USB hub mounting
- Power strip organization
- Quick-access ports

## Goals

- Clean desk surface
- Easy access to frequently used ports
- Hidden cable routing
- Modular so it can adapt as setup changes
